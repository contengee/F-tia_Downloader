# F-tia Downloader
Free Fantia Downloader

# Environment
OS: Linux

Python: 3.8.12

Using Python EEL GUI.

```$ cd python_env
$ pip install -r requirements.txt
```
or
```
$ conda env create --file run_env.yaml
```


# Run
```
$ python main.py
```

# Build run file(.exe)
```
$ python -m eel main.py web --onefile --noconsole
```
